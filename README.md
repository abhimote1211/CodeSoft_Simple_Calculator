

# CodeSoft Simple Calculator in C++  

## Description  
This is a simple command-line calculator built in C++ as part of **CodeSoft C++ Internship Task 2**.  
The calculator performs basic arithmetic operations:  

- Addition (`+`)  
- Subtraction (`-`)  
- Multiplication (`*`)  
- Division (`/`)  

It prompts the user to input two numbers and an operator, then displays the result.


2. **Compile the code**  
